# 5_Conditions

Τα condition στην python εχουν πολυ απλη δομή παρακάτω βλέπουμε παραδήγματα 

## Απλη if 

```Python
if <condition>:
    commands
```

```Python
if <condition>:
    commands
else:
    commands
```

## Εμφολευμένη if


```Python
if <condition>:
    commands
    if <condition2>:
        commands
    else:
        commands
else:
    commands
    if <condition3>:
        commands
    else:
        commands
```

## Συνθετη if 

```Python
if <condition>:
    commands
elif <condition2>:
    commands
```
